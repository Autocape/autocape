Automatic Minecraft cape generator.   

## Setup
Install dependencies using `npm install`.    
Set up server config using **config.json.example**, and save it as **config.json**.

## Developement
Run with automatic restarts on file changes:    
`npm run dev` or `nodemon .`

## Running in production
Start server using:    
`npm run` or `node .`
